Smart Tank: Automated Water Management System

IoT Android Kotlin MQTT Firebase ESP8266

Introduction

Tired of manual water level checks and wasted water? Our Smart Tank system offers an automated, intelligent solution for efficient water management across multiple tanks. It ensures optimal water levels, prevents overflow, and provides real-time control, all from your smartphone.

How It Works

The system utilizes three interconnected tanks: a Supply Reservoir (source), a Smart Tank (main monitored tank with sensors), and a Distribution Tank (for usage). An ESP8266 microcontroller continuously monitors the Smart Tank's level using an ultrasonic sensor. This data is wirelessly transmitted to a custom Android application.

Based on user-defined thresholds, the system automatically activates a submersible pump in the Supply Reservoir to fill the Smart Tank, maintaining precise water levels without manual intervention.

Key Features

  • Real-time Monitoring: View current water levels (percentage and depth) instantly via the Android app.
  • Automated Pump Control: Set desired fill levels; the pump manages itself.
  • Manual Override: Full manual control (ON/OFF) of the pump from the app.
  • Critical Level Alerts: Receive instant push notifications for low or full tank conditions.
  • Intuitive Mobile App: User-friendly interface for seamless control and data visualization.

Technology Stack

Hardware:
  • ESP8266 Microcontroller
  • Ultrasonic Sensor (HC-SR04)
  • Submersible Pump
Software & Cloud:
  • Android (Kotlin)
  • MQTT Protocol (hivemq.com)
  • Firebase Cloud Messaging (FCM)

Conclusion / Future Scope

This prototype demonstrates a robust solution for home, industrial, or agricultural water management, reducing manual effort and preventing resource waste. Future enhancements could include historical data analytics, multi-tank synchronization, and integration with smart home platforms.
