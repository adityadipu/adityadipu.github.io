Smart Cactus Pot: An IoT & Machine Learning Project

A solar-powered, intelligent plant system that monitors its own health and uses anomaly detection to identify critical events.

View on GitHub View Analysis
Back to All Projects

Project Overview

The Problem: Typical houseplants require constant manual checks. It's easy to forget to water them, or to water them improperly.

The Solution: I built a 100% solar-powered, autonomous pot that monitors its own environment in real-time. It streams data to a mobile app and, most importantly, uses machine learning to understand its own health and detect problems.

Technologies Used

ESP8266 Solar Panel DHT11 Soil Moisture Sensor CO2 Sensor Blynk IoT Python Pandas Scikit-learn C++ (Arduino)

The Build: Hardware & Data Flow

I assembled the system using an ESP8266 as the central brain. All sensors connect to it, and it runs on a C++ firmware to read data and send it to the cloud. A key feature is the solar panel and battery module, which makes the entire system self-sufficient and fully autonomous.

Live Data Dashboard

I used the Blynk IoT platform to create a live mobile dashboard. This allows for monitoring all sensor data (Temperature, Humidity, Light Level, Soil Moisture, and Air Quality) from anywhere in the world, streamed directly from the ESP8266.

Beyond Simple Rules: Intelligent Anomaly Detection

A simple rule like if soil < 50 isn't smart. It doesn't account for temperature, light, or other factors. I wanted the pot to learn its own 'normal' state. I first analyzed the sensor data to find relationships, like how temperature and humidity interact.

I then trained an Isolation Forest model (an anomaly detection algorithm) on the data. The model learned the plant's complex 'normal' state from all sensors combined.

The model can now successfully identify 'anomalous' data points. As shown in the plot below, the detected anomalies (red 'X's) perfectly correspond to the moments of critical soil dryness, proving the system can intelligently flag when it needs attention.

Future Vision

  • Close the Loop: Integrate the ML model to automatically trigger the water pump, creating a truly autonomous, self-watering system.
  • Custom App: Build a dedicated mobile app (e.g., in React Native or Flutter) to replace Blynk for a more tailored user experience.
